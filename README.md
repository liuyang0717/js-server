该项目仓库地点[GitHub](https://github.com/liuyang0717/itxia-web/tree/master/src/back-end/js-server)

## 项目简介
js-server主要使用express框架建立，具体方案请查看[designs](https://github.com/liuyang0717/js-server/tree/master/designs)文档。
